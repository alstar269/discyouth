# 찰떡소통 DISC 검사 (3D Animation Edition)

## 프로젝트 개요
10대 청소년을 위한 트렌디한 3D 애니메이션 감성의 DISC 성격 유형 검사 웹앱입니다. 입체적인 캐릭터와 깔끔한 UI를 통해 가독성과 시각적 즐거움을 동시에 제공합니다.

## 디자인 컨셉: "3D Claymorphism"
- **캐릭터 스타일**: 단순한 평면 아이콘이 아닌, 부드러운 그림자와 빛 반사가 들어간 3D 클레이 스타일 avatar.
- **타이포그래피**: 가독성을 최우선으로 고려. 제목은 포인트 있게, 본문은 깔끔한 `Pretendard` 서체 사용. (기존의 너무 굵은 폰트 지양)
- **컬러**: 부드러운 파스텔 톤과 생생한 포인트 컬러의 조합. 입체감을 주는 그라디언트 활용.
- **UI 요소**: '말랑말랑한' 느낌의 버튼과 카드 디자인 (Soft UI / Neumorphism 기법 가미).

## 가독성 개선 사항
- 폰트 두께 조절: 본문 텍스트의 가중치를 낮추고 행간을 넓혀 읽기 편하게 조정.
- 대비 최적화: 배경과 텍스트의 대비를 명확히 하여 모바일에서도 눈이 편안하도록 구성.

## 개발 계획
1.  **캐릭터 리디자인**: SVG Radial Gradient를 활용한 3D 입체 캐릭터 구현.
2.  **스타일 시트 전면 수정**: 3D 느낌의 쉐도우와 가독성 중심의 폰트 설정.
3.  **UI 밸런스 조정**: 전체적인 레이아웃을 더 여유롭고 깔끔하게 배치.
