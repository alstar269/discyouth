# 찰떡소통 DISC 검사 (Perfect Communication DISC Test)

## 프로젝트 개요
10대 청소년을 타겟으로 한 트렌디하고 직관적인 DISC 성격 유형 검사 웹앱입니다. 스마트폰 환경에 최적화된 UI/UX와 귀여운 캐릭터 일러스트를 통해 사용자에게 즐거운 경험을 제공합니다.

## 구현 기능 및 디자인 특징
- **프레임워크 프리**: 순수 HTML, CSS, JavaScript 및 Web Components 사용.
- **모바일 우선 디자인**: 손쉬운 터치 인터페이스와 부드러운 전환 효과.
- **현대적 CSS**: `oklch` 컬러 시스템, `container queries`, `:has()` 셀렉터 등을 활용한 세련된 스타일.
- **캐릭터 브랜딩**: 각 유형별 상징적인 캐릭터(사자, 토끼, 코알라, 부엉이) 배치.
- **인터랙티브 퀴즈**: 프로그레스 바와 슬라이드 애니메이션이 적용된 20문항 테스트.

## 개발 계획 및 단계
1.  **기본 구조 설정**: `index.html`, `style.css`, `main.js` 파일 생성.
2.  **디자인 시스템 구축**: 10대 감성에 맞는 활기찬 컬러와 타이포그래피 설정.
3.  **컴포넌트 개발**: 메인 앱 로직과 화면 전환을 담당하는 Web Component 구현.
4.  **데이터 통합**: 제공된 20개 질문과 결과 데이터 반영.
5.  **최종 검토 및 최적화**: 애니메이션 부드러움 및 모바일 반응성 확인.
